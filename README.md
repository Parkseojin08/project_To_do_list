# MINI_PROJECT_TO_DO_LIST
## 프로젝트 도구 및 기술 스택
Tool: vscode, DBeaver
SQL: postgreSQL
라이브러리: React
서버: node.js

## API
Swagger API 문서:

https://petstore.swagger.io/?url=https://raw.githubusercontent.com/Parkseojin08/project_To_do_list/main/server/swagger/swagger.yaml#/

## 사용법

1. 회원가입
   - 입력: 이름, 비밀번호, 이메일
    
2. 로그인
   - 입력: 이메일, 비밀번호

3. todo List 사용
   - 만들기 버튼: To Do List 수정
   - 검색: To Do List의 제목 검색
   - 천제확인 버튼: 모든 To Do List 호출
   - 체크 O 확인 버튼: 완료한 To Do List만 출
   - 체크 X 확인 버튼: 완료하지 않은 버튼만 호출
